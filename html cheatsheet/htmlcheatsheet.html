HTML Cheatsheet

HTML Tags

  • <div> - Defines a division or a section
  • <p> - Defines a paragraph
  • <ul> - Defines an unordered list
  • <ol> - Defines an ordered list
  • <li> - Defines a list item
  • <a> - Defines a hyperlink

HTML Attributes

  • id - Defines a unique identifier for an element
  • class - Defines one or more class names for an element
  • href - Defines the URL of a link
  • src - Defines the URL of an image
  • alt - Defines alternative text for an image

HTML Attributes

  • id - Defines a unique identifier for an element
  • class - Defines one or more class names for an element
  • href - Defines the URL of a link
  • src - Defines the URL of an image
  • alt - Defines alternative text for an image
  • title - Provides additional information about an element (tooltip)
  • style - Defines inline CSS styles for an element

Metadata Tags

  • <head> - Contains meta-information about the document
  • <title> - Sets the title of the document
  • <meta> - Provides metadata about the HTML document
  • <base> - Specifies the base URL/target for all relative URLs in a document

Multimedia Tags

  • <audio> - Embeds sound content in a document
  • <video> - Embeds video content in a document
  • <source> - Specifies multiple media resources for media elements
  • <track> - Specifies text tracks for video elements

Form-related Tags

  • <form> - Defines a form for user input
  • <input> - Defines an input field
  • <textarea> - Defines a multiline text input control
  • <select> - Defines a dropdown list
  • <button> - Defines a clickable button

Semantic HTML Tags

  • <header> - Represents introductory content
  • <footer> - Represents the footer of a document or a section
  • <nav> - Defines a set of navigation links
  • <article> - Represents independent, self-contained content
  • <section> - Represents a generic standalone section
